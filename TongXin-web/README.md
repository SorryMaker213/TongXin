# 仝心 TongXin - 前端项目

## 项目简介

这是仝心 TongXin 的前端应用，基于 Vue 3 + Vite + Element Plus。

## 安装依赖

```bash
cd TongXin-web
npm install
```

## 启动开发环境

```bash
npm run dev
```

启动后访问：

- http://localhost:5173

## 构建生产包

```bash
npm run build
```

## 预览生产包

```bash
npm run preview
```

## 目录结构

```text
TongXin-web/
├── public/
├── src/
├── index.html
├── package.json
└── vite.config.js
```

## 说明

- 接口默认通过 Vite 代理转发到 http://localhost:3000。
- 代理配置位于 `vite.config.js`。
