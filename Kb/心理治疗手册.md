# 第一部分：认知行为疗法（CBT）实用指南

## 1. CBT 核心理论

### 1.1 基本模型

**情境 → 认知（想法）→ 情绪 → 行为 → 生理反应**

CBT 认为：**并非事件本身导致情绪，而是你对事件的解读（认知）决定情绪与行为。**

### 1.2 核心观点

- 思维、情绪、行为相互影响、循环作用

- 功能不良的思维模式可被识别、修正

- 改变可通过练习逐步实现，而非顿悟

- 强调**当下问题**与**可操作技能**，而非童年追溯

## 2. 常见认知偏差（核心识别清单）

1. **非黑即白思维**：非好即坏，无中间地带

2. **灾难化**：小事放大为极端糟糕结果

3. **以偏概全**：单一事件推导普遍结论

4. **心理过滤**：只看负面，忽略正面证据

5. **否定正面**：将成功归为运气，贬低自我

6. **读心术**：主观臆断他人想法

7. **预测未来**：武断预判负面结果

8. **个人化**：无关事件归罪于自己

9. **应该陈述**：用“必须/应该”苛责自己或他人

10. **情绪推理**：“我感觉如此，所以就是真的”

## 3. CBT 标准流程（结构化步骤）

### 3.1 第一步：问题清单化

- 列出当前困扰：情绪、行为、人际、学业/工作

- 评估严重程度（0–10 分）

- 确定优先处理目标

### 3.2 第二步：识别自动思维

当强烈情绪出现时，自问三句：

- 刚才发生了什么？

- 我脑海中闪过什么念头？

- 这个想法让我产生了什么情绪？

### 3.3 第三步：检验与挑战思维

使用**苏格拉底提问**：

- 支持这个想法的证据是什么？反对的呢？

- 有没有更合理的解释？

- 最坏/最好/最可能结果分别是什么？

- 这种想法对我有帮助吗？

- 如果朋友这样想，我会怎么劝他？

### 3.4 第四步：建立平衡思维

用更现实、温和、功能性的陈述替代极端思维。

示例：

- 原思维：我彻底失败了

- 平衡思维：这次没做好，但不代表我一无是处，我可以改进。

### 3.5 第五步：行为实验

用行动验证想法，例如：

- 害怕被拒绝 → 主动发起一次对话

- 害怕出错 → 刻意允许自己犯小错

- 回避社交 → 逐步增加轻度社交活动

### 3.6 第六步：复发预防

- 识别高危情境

- 制定应对预案

- 巩固技能，形成长期习惯

## 4. 核心技术工具

### 4.1 思维记录表（经典 CBT 工作表）

1. 情境（时间、地点、事件）

2. 情绪（名称+强度 0–100%）

3. 自动思维

4. 认知偏差类型

5. 支持/反对证据

6. 平衡替代思维

7. 情绪再评分

### 4.2 行为激活（针对抑郁、低动机）

- 安排小而具体的活动

- 按愉悦感/掌控感评分

- 逐步提升活动复杂度

- 打破“不做事→更抑郁→更不做事”循环

### 4.3 暴露疗法（针对焦虑、恐惧、强迫）

- 分级暴露恐惧刺激

- 不使用安全行为

- 延长停留至焦虑自然下降

- 重复练习形成新习得

### 4.4 放松与呼吸训练

- 腹式呼吸

- 渐进式肌肉放松

- 正念停留技术

用于降低生理唤醒，辅助认知调整

### 4.5 问题解决训练

1. 定义问题

2. 列出所有可能方案

3. 评估利弊

4. 选择执行

5. 复盘效果

## 5. 适用范围

- 抑郁障碍

- 各类焦虑障碍（社交恐惧、惊恐、广泛性焦虑、强迫症）

- 创伤后应激

- 失眠

- 冲动控制问题

- 压力管理、情绪调节

- 亲子、伴侣关系冲突

## 6. 不适用/需谨慎

- 严重精神病性症状（幻觉、妄想）需先药物稳定

- 急性自伤/自杀高风险需优先危机干预

- 严重认知损害、痴呆

- 完全拒绝合作、无改变动机者

# 第二部分：心理急救（PFA）操作手册

## 1. 心理急救核心原则

1. **安全第一**：先保证身体安全，再进行心理支持

2. **不强迫**：不强迫讲述、不强迫情绪表达

3. **不评判**：不评价对错、不道德说教

4. **简洁务实**：提供实际帮助，而非长篇心理分析

5. **稳定优先**：先稳住情绪，再处理信息

6. **尊重文化与隐私**

## 2. 适用场景

- 自然灾害、事故、暴力事件

- 丧亲、重伤、目击创伤

- 公共突发事件

- 校园危机、职场冲突升级

- 急性惊恐、情绪崩溃

## 3. 标准步骤：LOOK → LISTEN → LINK

### 3.1 观察（Look）

- 观察环境是否安全

- 观察当事人状态：是否受伤、极度恐慌、呆滞、愤怒、自伤风险

- 识别高危信号：自杀言语、冲动攻击、严重解离

### 3.2 倾听（Listen）

- 温和自我介绍，保持适当距离

- 用开放式问题：你现在感觉怎么样？需要什么帮助？

- 允许沉默，不催促讲述细节

- 简单共情：这确实很难受、换谁都会害怕

### 3.3 联结（Link）

- 提供实际帮助：水、休息、联系家人

- 链接资源：医疗、家属、专业心理/危机热线

- 提供清晰、简短的信息，减少不确定性

## 4. 现场核心操作技能

### 4.1 安抚与稳定

- 语气平稳、语速放缓

- 简单重复保证：你现在是安全的

- 引导慢呼吸：吸气4秒→屏息4秒→呼气6秒

- 避免过度刺激（强光、噪音、拥挤）

### 4.2 实际需求优先

优先解决：

- 冷、热、渴、痛

- 失联、找不到家人

- 不知所措、无下一步计划

### 4.3 正常化反应

告诉当事人：

- 恐惧、失眠、闪回、易怒都是**正常创伤反应**

- 多数人会在数天到数周内逐步缓解

- 不代表“疯了”或“脆弱”

## 5. 绝对禁忌（严禁做的事）

- 不追问创伤细节：“当时发生了什么？你怎么不跑？”

- 不说“别难过”“别哭了”“要坚强”

- 不做虚假保证：“一切都会马上好起来”

- 不比较痛苦：“别人比你更惨”

- 不随意诊断、不贴标签

- 不强迫放松、不强行拥抱或肢体接触

- 不传播现场信息、不拍照录像

## 6. 高危情况识别与转介

出现以下情况需**立即转介专业危机干预**：

- 明确自杀/伤人计划或行为

- 严重解离、意识混乱

- 持续无法安抚、极度亢奋或木僵

- 严重躯体症状伴心理应激

- 儿童、老人、孕妇等高脆弱群体

## 7. 常用支持话术（可直接使用）

- 我在这里陪着你，你现在是安全的。

- 你可以慢慢说，不想说也没关系。

- 你的反应很正常，很多人经历这些都会这样。

- 我会帮你联系家人/医生。

- 我们一起一步一步来。

## 8. 后续自我照顾建议（给当事人）

- 保持规律作息、饮食

- 避免酒精、药物麻痹情绪

- 减少刷创伤相关新闻

- 向信任的人倾诉

- 若失眠、噩梦、惊恐持续超过两周，寻求专业心理咨询
